# Covid-19-Forecasting
COVID-19 time series forecasting project using Linear Regression, Random Forest, XGBoost, ARIMA, SARIMA, Prophet, and LSTM models. SARIMA achieved the best forecasting performance for predicting daily COVID-19 cases in India.
# COVID-19 Case Forecasting Using Time Series Analysis

Developed a time-series forecasting solution to predict daily COVID-19 confirmed cases in India using data from the Johns Hopkins COVID-19 repository. Performed data preprocessing, feature engineering, trend analysis, and model evaluation using multiple forecasting approaches including Linear Regression, Random Forest, XGBoost, ARIMA, SARIMA, Prophet, and LSTM.

**Key Highlights:**
- Time Series Forecasting & Trend Analysis
- Feature Engineering (Lag Features, Rolling Averages, Growth Rates)
- Machine Learning, Statistical & Deep Learning Models
- Model Performance Comparison using MAE, RMSE, MAPE, sMAPE, and R²
- Best Model: **SARIMA** (Lowest Forecasting Error and Highest R² Score)
